GitHub입문

Pull Request를 보내봐요!

It's a really great practice for git

이 사이트는 제이펍에서 출간된 번역서 GitHub 입문과 관련된 사이트입니다.

책을 구입하고 이용해주시면 감사하겠습니다.

이 페이지의 레퍼지토리는 상단의 "Give me Pull Request"를 누르면 들어갑니다.

주의 사항

책과 관련되지 않은 일 또는 비방 글은 관리자의 판단으로 게시되지 않을 수 있습니다.

Pull Request를 제대로 보내시면 리포지토리 권한을 드립니다. 보내는 것 이외에도 받는 연습도 해보세요.

물론 권한을 받아도 부적절한 행위를 한다면 관리자의 판단으로 저장소 접근 권한을 없앨 수 있습니다.

이 페이지는 역자와 권한을 받은 사람들이 관리합니다. 따라서 Pull Request를 보내도 곧바로 반영되지 않을 수 있습니다.
